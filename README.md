Install
--------

[base](http://qiita.com/items/708c77b0986674a862e4)

```bash
cd /tmp
mkdir -p ~/local/bin
wget https://github.com/downloads/D-Programming-Language/dmd/dmd.2.060.zip
unzip dmd.2.060.zip
mv dmd2 ~/local/bin
echo 'export PATH="$HOME/local/bin/dmd2/linux/bin64:$PATH"' >> ~/.bashrc
source ~/.bashrc
```
